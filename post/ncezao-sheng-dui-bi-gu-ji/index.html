NCE：噪声对比估计

发布于 2020-07-04 | 分类于 ML 、 NLP | 6分钟 | 1073字数

SoftmaxSoftmaxSoftmax 带来的归一化难题

SoftmaxSoftmaxSoftmax 的计算表达式如下：

P(xi)=Softmax(xi)=exi∑jexjP(x_i) = Softmax(x_i) = \frac{e^{x_i}}{\sum_{j}e^{x_j}} P(xi​)=Softmax(xi​)=∑j​exj​exi​​

其中 Z=∑jexjZ = \sum_{j}e^{x_j}Z=∑j​exj​ 是保证概率和为 1 的归一化常数，也被称为“配分函数”，注意它是指数和的形式，当项数巨大时，将给计算带来不小的负担。
在利用 SoftmaxSoftmaxSoftmax 对概率进行建模时，常用的损失函数包括交叉熵和负对数似然，这两种损失函数均无法避免对配分函数的计算，因此不是那么地适合用于项数巨大的分类任务。
NCE 是为了解决这种难题而提出来的一种方法，它通过引入噪声将多分类问题转变为二分类问题，虽然目标形式发生了转变，但是却能达到与原来目标参数训练相近的效果。

负对数似然下的损失及其梯度

设数据集样本分布为 Pd(x)P_d(x)Pd​(x)，模型给出的分布为 Pθ(x)=efθ(x)ZP_{\theta}(x) = \frac{e^{f_{\theta}(x)}}{Z}Pθ​(x)=Zefθ​(x)​，其负对数似然损失（交叉熵损失在此与之相似）如下：

loss1=−Ex∼Pd(x)[logPθ(x)]loss_1 = -E_{x \sim P_d(x)}[logP_{\theta}(x)] loss1​=−Ex∼Pd​(x)​[logPθ​(x)]

化简可得：

loss1=−Ex∼Pd(x)[fθ(x)]+logZloss_1 = -E_{x \sim P_d(x)}[f_{\theta}(x)] + logZ loss1​=−Ex∼Pd​(x)​[fθ​(x)]+logZ

我们来看一下 loss1loss_1loss1​ 对参数 θ\thetaθ 的梯度：

∂loss1∂θ=−Ex∼Pd(x)[∂fθ(x)∂θ]+1Z∂Z∂θ\frac{\partial loss_1}{\partial \theta} = -E_{x \sim P_d(x)}[\frac{\partial f_{\theta}(x)}{\partial \theta}]+\frac{1}{Z}\frac{\partial Z}{\partial \theta} ∂θ∂loss1​​=−Ex∼Pd​(x)​[∂θ∂fθ​(x)​]+Z1​∂θ∂Z​

化简可得：

∂loss1∂θ=−Ex∼Pd(x)[∂fθ(x)∂θ]+Ex∼Pθ(x)[∂fθ(x)∂θ]\frac{\partial loss_1}{\partial \theta} = -E_{x \sim P_d(x)}[\frac{\partial f_{\theta}(x)}{\partial \theta}] + E_{x \sim P_{\theta}(x)}[\frac{\partial f_{\theta}(x)}{\partial \theta}] ∂θ∂loss1​​=−Ex∼Pd​(x)​[∂θ∂fθ​(x)​]+Ex∼Pθ​(x)​[∂θ∂fθ​(x)​]

第一个期望只能通过采样获得，第二个期望可以通过计算或者采样获得，但是均无法避免计算配分函数。

NCE 方法

引入噪声分布 Pn(x)P_n(x)Pn​(x)，设其强度为 kkk，数据自以下分布中采样：

P(x)=1k+1(Pd(x)+kPn(x))P(x) = \frac{1}{k+1}(P_d(x)+kP_n(x)) P(x)=k+11​(Pd​(x)+kPn​(x))

设自训练数据中采样的样本为正样本（d=1d = 1d=1），自噪声分布中采样的样本为负样本（d=0d = 0d=0），令模型判别正样本的似然为：

P+(d=1∣x)=Pθ(x)Pθ(x)+kPn(x)P_{+}(d = 1|x) = \frac{P_{\theta}(x)}{P_{\theta}(x)+kP_n(x)} P+​(d=1∣x)=Pθ​(x)+kPn​(x)Pθ​(x)​

化简可得：

P+(d=1∣x)=11+ZkPn(x)e−fθ(x)P_{+}(d = 1|x) = \frac{1}{1+ZkP_n(x)e^{-f_{\theta}(x)}} P+​(d=1∣x)=1+ZkPn​(x)e−fθ​(x)1​

对于上式中的 ZZZ，不再由归一化生成，而是作为可训练的参数由训练生成。
令 ZkPn(x)=eγz(x)ZkP_n(x) = e^{\gamma_{z}(x)}ZkPn​(x)=eγz​(x)，可得：

P+(d=1∣x)=11+e−fθ(x)+γz(x)=σ(fθ(x)−γz(x))P_{+}(d = 1|x) = \frac{1}{1+e^{-f_{\theta}(x)+\gamma_{z}(x)}} = \sigma(f_{\theta}(x)-\gamma_{z}(x)) P+​(d=1∣x)=1+e−fθ​(x)+γz​(x)1​=σ(fθ​(x)−γz​(x))

至此，通过引入噪声得到了一个二分类问题，其二元交叉熵如下：

loss2=−E(x,d)∼P(x)[dlogP+(d=1∣x)+(1−d)log(1−P+(d=1∣x))]loss_2 = -E_{(x, d) \sim P(x)}[dlogP_{+}(d = 1|x)+(1-d)log(1-P_{+}(d = 1|x))] loss2​=−E(x,d)∼P(x)​[dlogP+​(d=1∣x)+(1−d)log(1−P+​(d=1∣x))]

化简可得：

loss2=−1k+1(Ex∼Pd(x)[logP+(d=1∣x)]+kEx∼Pn(x)[log(1−P+(d=1∣x))])loss_2 = -\frac{1}{k+1}(E_{x \sim P_d(x)}[logP_{+}(d = 1|x)]+kE_{x \sim P_n(x)}[log(1-P_{+}(d = 1|x))]) loss2​=−k+11​(Ex∼Pd​(x)​[logP+​(d=1∣x)]+kEx∼Pn​(x)​[log(1−P+​(d=1∣x))])

放下前面的常系数，两个期望均通过采样获得，则有：

loss2=−(1m∑xi∼Pdmlogσ(fθ(xi)−γz(xi))+kn∑xj∼Pnnlog(1−σ(fθ(xj)−γz(xj))))loss_2 = -(\frac{1}{m}\sum_{x_i \sim P_d}^{m}log\sigma(f_{\theta}(x_i)-\gamma_{z}(x_i))+\frac{k}{n}\sum_{x_j \sim P_n}^{n}log(1-\sigma(f_{\theta}(x_j)-\gamma_{z}(x_j)))) loss2​=−(m1​xi​∼Pd​∑m​logσ(fθ​(xi​)−γz​(xi​))+nk​xj​∼Pn​∑n​log(1−σ(fθ​(xj​)−γz​(xj​))))

运用梯度下降优化参数即可。
