    SoftmaxSoftmaxSoftmax 的归一化难题

    SoftmaxSoftmaxSoftmax 的计算表达式如下：

    P(xi)=Softmax(xi)=exi∑jexjP(x_i) = Softmax(x_i) = \frac{e^{x_i}}{\sum_{j}e^{x_j}} P(xi​)=Softmax(xi​)=∑j​exj​exi​​

    其中 Z=∑jexjZ = \sum_{j}e^{x_j}Z=∑j​exj​ 是保证概率和为 1 的归一化常数，也被称为“配分函数”，注意它是指数和的形式，当项数巨大时，将给计算带来不小的负担。
    在利用 SoftmaxSoftmaxSoftmax 对概率进行建模时，常用的损失函数包括交叉熵和负对数似然，这两种损失函数均无法避免对配分函数的计算，因此不是那么地适合用于项数巨大的分类任务。
    NCE 是为了解决这种难题而提出来的一种方法，它通过引入噪声将多分类问题转变为二分类问题，虽然目标形式发生了转变，但是却能达到与原来目标参数训练相近的效果。

    负对数似然损失

    设数据集样本分布为 Pd(x)P_d(x)Pd​(x)，模型给出的分布为 Pθ(x)=efθ(x)ZP_{\theta}(x) = \frac{e^{f_{\theta}(x)}}{Z}Pθ​(x)=Zefθ​(x)​，其负对数似然损失（交叉熵损失在此与之相似）如下：

    loss1=−Ex∼Pd(x)[logPθ(x)]loss_1 = -E_{x \sim P_d(x)}[logP_{\theta}(x)] loss1​=−Ex∼Pd​(x)​[logPθ​(x)]

    化简可得：

    loss1=−Ex∼Pd(x)[fθ(x)]+logZloss_1 = -E_{x \sim P_d(x)}[f_{\theta}(x)] + logZ loss1​=−Ex∼Pd​(x)​[fθ​(x)]+logZ

    我们来看一下 loss1loss_1loss1​ 对参数 θ\thetaθ 的梯度：

    ∂loss1∂θ=−Ex∼Pd(x)[∂fθ(x)∂θ]+1Z∂Z∂θ\frac{\partial loss_1}{\partial \theta} = -E_{x \sim P_d(x)}[\frac{\partial f_{\theta}(x)}{\partial \theta}]+\frac{1}{Z}\frac{\partial Z}{\partial \theta} ∂θ∂loss1​​=−Ex∼Pd​(x)​[∂θ∂fθ​(x)​]+Z1​∂θ∂Z​

    化简可得：

    ∂loss1∂θ=−Ex∼Pd(x)[∂fθ(x)∂θ]+Ex∼Pθ(x)[∂fθ(x)∂θ]\frac{\partial loss_1}{\partial \theta} = -E_{x \sim P_d(x)}[\frac{\partial f_{\theta}(x)}{\partial \theta}] + E_{x \sim P_{\theta}(x)}[\frac{\partial f_{\theta}(x)}{\partial \theta}] ∂θ∂loss1​​=−Ex∼Pd​(x)​[∂θ∂fθ​(x)​]+Ex∼Pθ​(x)​[∂θ∂fθ​(x)​]

    第一个期望只能通过采样获得，第二个期望可以通过计算或者采样获得，但是均无法避免计算配分函数。

    NCE 方法

    引入噪声分布 Pn(x)P_n(x)Pn​(x)，设其强度为 kkk，数据自以下分布中采样：

    P(x)=1k+1(Pd(x)+kPn(x))P(x) = \frac{1}{k+1}(P_d(x)+kP_n(x)) P(x)=k+11​(Pd​(x)+kPn​(x))

    设自训练数据中采样的样本为正样本（d=1d = 1d=1），自噪声分布中采样的样本为负样本（d=0d = 0d=0），令模型判别正样本的似然为：

    P+(d=1∣x)=Pθ(x)Pθ(x)+kPn(x)P_{+}(d = 1|x) = \frac{P_{\theta}(x)}{P_{\theta}(x)+kP_n(x)} P+​(d=1∣x)=Pθ​(x)+kPn​(x)Pθ​(x)​

    化简可得：

    P+(d=1∣x)=11+ZkPn(x)e−fθ(x)P_{+}(d = 1|x) = \frac{1}{1+ZkP_n(x)e^{-f_{\theta}(x)}} P+​(d=1∣x)=1+ZkPn​(x)e−fθ​(x)1​

    对于上式中的 ZZZ，不再由归一化生成，而是作为可训练的参数由训练生成，或者直接设置为常数 1，这要求模型具有“self-normalize”的能力。
    令 ZkPn(x)=eγz(x)ZkP_n(x) = e^{\gamma_{z}(x)}ZkPn​(x)=eγz​(x)，可得：

    P+(d=1∣x)=11+e−fθ(x)+γz(x)=σ(fθ(x)−γz(x))P_{+}(d = 1|x) = \frac{1}{1+e^{-f_{\theta}(x)+\gamma_{z}(x)}} = \sigma(f_{\theta}(x)-\gamma_{z}(x)) P+​(d=1∣x)=1+e−fθ​(x)+γz​(x)1​=σ(fθ​(x)−γz​(x))

    至此，通过引入噪声得到了一个二分类问题，其二元交叉熵如下：

    loss2=−E(x,d)∼P(x)[dlogP+(d=1∣x)+(1−d)log(1−P+(d=1∣x))]loss_2 = -E_{(x, d) \sim P(x)}[dlogP_{+}(d = 1|x)+(1-d)log(1-P_{+}(d = 1|x))] loss2​=−E(x,d)∼P(x)​[dlogP+​(d=1∣x)+(1−d)log(1−P+​(d=1∣x))]

    化简可得：

    loss2=−1k+1(Ex∼Pd(x)[logP+(d=1∣x)]+kEx∼Pn(x)[log(1−P+(d=1∣x))])loss_2 = -\frac{1}{k+1}(E_{x \sim P_d(x)}[logP_{+}(d = 1|x)]+kE_{x \sim P_n(x)}[log(1-P_{+}(d = 1|x))]) loss2​=−k+11​(Ex∼Pd​(x)​[logP+​(d=1∣x)]+kEx∼Pn​(x)​[log(1−P+​(d=1∣x))])

    丢弃前面的常系数：

    loss2=−Ex∼Pd(x)[logP+(d=1∣x)]−kEx∼Pn(x)[log(1−P+(d=1∣x))]loss_2 = -E_{x \sim P_d(x)}[logP_{+}(d = 1|x)]-kE_{x \sim P_n(x)}[log(1-P_{+}(d = 1|x))] loss2​=−Ex∼Pd​(x)​[logP+​(d=1∣x)]−kEx∼Pn​(x)​[log(1−P+​(d=1∣x))]

    两个期望均通过采样获得，则有：

    loss2=−1m∑xi∼Pdmlogσ(fθ(xi)−γz(xi))−kn∑xj∼Pnnlog(1−σ(fθ(xj)−γz(xj)))loss_2 = -\frac{1}{m}\sum_{x_i \sim P_d}^{m}log\sigma(f_{\theta}(x_i)-\gamma_{z}(x_i))-\frac{k}{n}\sum_{x_j \sim P_n}^{n}log(1-\sigma(f_{\theta}(x_j)-\gamma_{z}(x_j))) loss2​=−m1​xi​∼Pd​∑m​logσ(fθ​(xi​)−γz​(xi​))−nk​xj​∼Pn​∑n​log(1−σ(fθ​(xj​)−γz​(xj​)))

    运用梯度下降优化参数即可。

    目标等价性

    NCE 方法所得的参数 θ\thetaθ 与原负对数似然损失所得的参数具有一致性，来看一下 NCE 损失对 θ\thetaθ 的梯度：

    ∂loss2∂θ=−Ex∼Pd(x)[(1−P+(x))∂fθ(x)∂θ]+kEx∼Pn(x)[P+(x)∂fθ(x)∂θ]\frac{\partial loss_2}{\partial \theta} = -E_{x \sim P_d(x)}[(1-P_{+}(x))\frac{\partial f_{\theta}(x)}{\partial \theta}]+kE_{x \sim P_n(x)}[P_{+}(x)\frac{\partial f_{\theta}(x)}{\partial \theta}] ∂θ∂loss2​​=−Ex∼Pd​(x)​[(1−P+​(x))∂θ∂fθ​(x)​]+kEx∼Pn​(x)​[P+​(x)∂θ∂fθ​(x)​]

    整理可得：

    ∂loss2∂θ=−Ex∼Pd(x)[kPn(x)Pθ(x)+kPn(x)∂fθ(x)∂θ]+Ex∼Pθ(x)[kPn(x)Pθ(x)+kPn(x)∂fθ(x)∂θ]\frac{\partial loss_2}{\partial \theta} = -E_{x \sim P_d(x)}[\frac{kP_n(x)}{P_{\theta}(x)+kP_n(x)}\frac{\partial f_{\theta}(x)}{\partial \theta}]+E_{x \sim P_{\theta}(x)}[\frac{kP_n(x)}{P_{\theta}(x)+kP_n(x)}\frac{\partial f_{\theta}(x)}{\partial \theta}] ∂θ∂loss2​​=−Ex∼Pd​(x)​[Pθ​(x)+kPn​(x)kPn​(x)​∂θ∂fθ​(x)​]+Ex∼Pθ​(x)​[Pθ​(x)+kPn​(x)kPn​(x)​∂θ∂fθ​(x)​]

    显然：

    lim⁡k→+∞∂loss2∂θ=∂loss1∂θ\lim_{k \rightarrow +\infty}\frac{\partial loss_2}{\partial \theta} = \frac{\partial loss_1}{\partial \theta} k→+∞lim​∂θ∂loss2​​=∂θ∂loss1​​

    上式表明，在噪声强度足够大的情况下，利用 NCE 方法对 θ\thetaθ 进行参数更新与原来利用负对数似然进行参数更新的效果是近似相等的。

    NCE 与负采样技术

    有声明称负采样技术源于 NCE 方法，最为有名的应该是 word2vec 这一基础 NLP 词嵌入技术。
    word2vec 以两个词向量的内积表示两个词的相似性：

    sim(wi,wj)=<vi,vj>sim(w_i, w_j) = <v_i, v_j> sim(wi​,wj​)=<vi​,vj​>

    在以 wjw_jwj​ 为上下文环境时，wiw_iwi​ 为中心词概率可以表示为：

    P(wi∣wj)=e<vi,vj>∑ke<vk,vj>=e<vi,vj>ZjP(w_i|w_j) = \frac{e^{<v_i, v_j>}}{\sum_{k}e^{<v_k, v_j>}} = \frac{e^{<v_i, v_j>}}{Z_j} P(wi​∣wj​)=∑k​e<vk​,vj​>e<vi​,vj​>​=Zj​e<vi​,vj​>​

    以采样自数据集分布 P~(wi∣wj)\tilde{P}(w_i|w_j)P~(wi​∣wj​) 的样本为正例，以各个词的词频 P~(wi)\tilde{P}(w_i)P~(wi​) 为噪声分布进行负采样，正例似然函数为：

    P+(d=1∣wi,wj)=σ(<vi,vj>−γz)P_{+}(d = 1|w_i, w_j) = \sigma(<v_i, v_j>-\gamma_z) P+​(d=1∣wi​,wj​)=σ(<vi​,vj​>−γz​)

    word2vec 中的负采样并非完全是 NCE 的形式，它需要在此人为设置使得 γz≈0\gamma_z \approx 0γz​≈0，则可得损失函数：

    lossj=−∑wiP~(wi∣wj)logσ(<vi,vj>)−∑wiP~(wi)log(1−σ(<vi,vj>))loss_j = -\sum_{w_i}\tilde{P}(w_i|w_j)log\sigma(<v_i, v_j>)-\sum_{w_i}\tilde{P}(w_i)log(1-\sigma(<v_i, v_j>)) lossj​=−wi​∑​P~(wi​∣wj​)logσ(<vi​,vj​>)−wi​∑​P~(wi​)log(1−σ(<vi​,vj​>))

    这里为了方便将噪声强度设为 1 。在人为设置 γz≈0\gamma_z \approx 0γz​≈0 之后，可得：

    Zj≈1P~(wi)Z_j \approx \frac{1}{\tilde{P}(w_i)} Zj​≈P~(wi​)1​

    由优化后最终的结果：

    P(wi∣wj)≈P~(wi∣wj)P(w_i|w_j) \approx \tilde{P}(w_i|w_j) P(wi​∣wj​)≈P~(wi​∣wj​)

    可得：

    <vi,vj>≈logP~(wi∣wj)P~(wi)=I(wi,wj)<v_i, v_j> \approx log\frac{\tilde{P}(w_i|w_j)}{\tilde{P}(w_i)} = I(w_i, w_j) <vi​,vj​>≈logP~(wi​)P~(wi​∣wj​)​=I(wi​,wj​)

    式中 I(wi,wj)I(w_i, w_j)I(wi​,wj​) 表示两词的点互信息。上式表明，word2vec 中的负采样技术使得两个词向量的内积被建模成了两个词的点互信息，这种双向的信息联系要比单向的条件概率 P(wi∣wj)P(w_i|w_j)P(wi​∣wj​) 更能反映词之间的语义关系。
